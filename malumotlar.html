HTML asosiy tushunchalari.

Elementlar.

Element - ochuvchi tag, content va yopuvchi tagdan iborat struktura.

Bo'sh element(Empty) - Faqatgina ochuvchi tagdan iborat bo'lgan elementlar.

Ichki element(Nested) - Boshqa element ichida joylashgan element.

Atributlar.

Atribut - element haqida qo'shimcha ma'lumot beruvchi maxsus so'zlar.

Styles.

Satrli stillar (Inline styles) - taglani ichida qo'llaniladi.

Ichki stillar (Internal styles) - head qismida style tagi ichida qo'llaniladi.

Tashqi stillar (External styles) - alohida papkada qo'llaniladi.

Ilovalar.

Ichki ilovalar(internal links)-fayllarga o'tish

Ichki ilova

Tashqi ilovalar(external links)-url manzillarga o'tish.

Tashqi ilova

Block va inline elementlar.

Block elementlari har doim yangi qatordan boshlanadi.Widthni maksimal ravishda to'ldiradi.

Inline elementlari o'ziga yetarli joyni oladi,yangi qatordan boshlanmaydi.

Block element ichida inline element yaratib bo'ladi lekin inline element ichida block element yaratib bo'lmaydi.
